Welcome to Boom Boom world.

This is a game to destroy the blocks by shooting bomb with cannons.
User can play the game in first person view.
User can challenge each stage and move on to the next stage.

How to Play

1. Press the START button.
2. When you click the screen, the bomb will fly from the center of the screen.
3. You have to hit some specific boxes to clear.
4. The number of balls is limited.
5. If you click the mouse for a while, then the speed of the ball will increase and make you to clear the game easily.
6. Press esc button to to exit the game.

Enjoy the game :)
